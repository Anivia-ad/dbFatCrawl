## Project Overview

豆瓣肥虫数据分析网站是一个 Python 课程项目，用 Flask 提供 Web 界面，用 SQLite 保存电影和图书作品数据，并通过 BeautifulSoup、requests、pandas、matplotlib 和 wordcloud 完成公开数据爬取、手工维护、CSV 导入导出、筛选检索、图表分析、作品对比和 HTML 报告生成。

## Quick Reference

- **Tech Stack:** Python 3.11+、Flask 3、SQLite、requests、BeautifulSoup4、pandas、matplotlib、wordcloud、pytest、uv
- **Entry Point:** `douban_fatworm:create_app`，本地命令为 `uv run flask --app douban_fatworm run`
- **Architecture Pattern:** 应用工厂 + Flask Blueprint + 模块化业务函数 + SQLite Repository 风格数据访问
- **Database:** `instance/douban_fatworm.sqlite3`，核心表为 `works`
- **Generated Runtime Paths:** `uploads/`、`exports/`、`reports/`、`src/douban_fatworm/static/generated/`

## Getting Started

### Prerequisites

- Python 3.11+
- uv
- Windows PowerShell 下建议先执行 `[Console]::OutputEncoding = [System.Text.Encoding]::UTF8`

### Setup

```powershell
[Console]::OutputEncoding = [System.Text.Encoding]::UTF8
uv sync --extra dev
```

### Run Locally

```powershell
uv run flask --app douban_fatworm run
```

### Run Tests

```powershell
uv run pytest
```
